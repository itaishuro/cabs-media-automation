Media Server Log Cleanup Script – Documentation

Purpose:
Automatically clears media server logs to prevent the system from filling up and interrupting media playback.

Location:

Task is set up in Task Scheduler on the media server.

How to View/Modify the Script:

Open Task Scheduler.

Look for the task named: Clear Media Server Logs (or similar).

Right-click → Properties → Actions tab.

The script or command line used to clear the logs will be visible here.

Schedule:

Runs every two weeks.

Ensures the process runs automatically without manual intervention.
